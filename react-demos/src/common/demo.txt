  <h3>Smart Editor Context Menu</h3>

        <p>Context menus allow to execute quick actions. A different set of actions will appear in the Context menu
            depending on the current selection. The Context menu is customized for the this demo with additional actions
            depending on the click target.
        </p>

        <p>To open the context menu simply set the <b>contextMenu</b> property to enable and right click via the mouse
            somewhere inside the Editor. A different menu options list will appear depending on the click target.</p>

        <p>
            The context menu is available for the following type of elements:
        </p>
